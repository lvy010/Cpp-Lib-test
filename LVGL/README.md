# LVGL 样式测试项目

这是一个演示 LVGL 样式系统的完整示例项目，展示了各种样式功能：

## 功能特性

- ✅ **基础样式**：蓝色按钮样式
- ✅ **状态样式**：按压状态深蓝色
- ✅ **层叠样式**：红色变体按钮
- ✅ **部件样式**：滑块红色圆形旋钮
- ✅ **本地样式**：橙色按钮直接设置
- ✅ **全局样式**：字体和颜色继承

## 快速开始

### 1. 安装依赖

```bash
make install-deps
```

### 2. 编译项目

```bash
make
```

### 3. 运行测试

```bash
make run
```

或者直接运行：

```bash
./lvgl_style_test
```

## 项目结构

```
.
├── main.c          # 主程序入口
├── styles.c        # 样式定义和界面创建
├── Makefile        # 编译配置
└── README.md       # 项目说明
```

## 样式演示

1. **按钮1**：基础蓝色样式 + 按压状态
2. **按钮2**：红色变体样式（层叠样式示例）
3. **按钮3**：本地样式（橙色，20px圆角）
4. **滑块**：红色圆形旋钮
5. **进度条**：默认样式

## 清理

```bash
make clean
```

## 注意事项

- 这是一个模拟环境的 LVGL 测试程序
- 实际显示需要配置真实的显示驱动
